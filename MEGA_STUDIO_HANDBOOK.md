## ðŸ“¦ Installation & Sync

To install this system on a new machine:

1.  **Clone the Repo:**
    ```bash
    git clone https://github.com/Aeon-Design/Flutter-Studio.git C:\Users\YourName\Flutter-Studio
    ```

2.  **Run Setup Script:**
    Execute the `Setup-New-PC.ps1` script (included in root).
    This will:
    *   Restore `.agent` and `.gemini` folders.
    *   Install VS Code, Flutter, Android Studio, Git.
    *   Configure VS Code settings for agent visibility.

## ðŸ”„ Synchronization

To sync changes between your local machine and this repo:
1.  Run `Sync-To-GitHub.ps1`
2.  It mirrors your local agents to the repo folder.
3.  It commits and pushes changes to GitHub automatically.
